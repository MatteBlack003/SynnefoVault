# Synnefo Solutions - Mock Exams Platform

Welcome to the Synnefo Solutions Mock Exams repository! This repository uses **GitHub Pages** to host a fast, simple, and beautifully designed single-page website where students can access mock exam papers for all our training departments.

## 👨‍🏫 Instructions for Team Leads

As a team lead, you are responsible for uploading your department's mock exams. It takes **just 1 easy step**. Note: You don't even need the command line; you can do this directly in the GitHub web interface!

### Upload the Question Paper
1. Open this repository on GitHub.
2. Navigate into your department's folder (e.g., `networking`).
3. Click on **Add File** -> **Upload files**.
4. Upload your PDF or assignment file and click **Commit changes**.

**🎉 You're done!** 
A GitHub auto-updater robot will see your file and update the `index.html` website automatically so students can see it. It will take ~1 minute to show up on the live website.

---

## 🚀 How to Setup GitHub Pages (For the Admin)
1. Push this repository to Synnefo Solutions's GitHub account.
2. Go to the repository **Settings**.
3. On the left sidebar, click on **Pages**.
4. Under "Build and deployment", set the source to **Deploy from a branch**.
5. Select the `main` (or `master`) branch. If the root contains other items, you might define the URL path to point to the `/synnefo-mock-exams` folder.
6. Wait a minute, and your platform is live!
